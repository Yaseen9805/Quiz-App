# Quiz App 🧠

A simple interactive Quiz Application built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Multiple-choice questions
* Score tracking
* Timer for each question
* Automatic question progression
* Restart quiz functionality
* High score stored using localStorage

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 🎮 How It Works

1. A question is displayed with 4 options
2. A timer (10 seconds) starts for each question
3. User selects an answer:

   * If correct → score increases
   * Moves to next question
4. If time runs out:

   * Automatically moves to next question
5. After last question:

   * Final score is displayed
   * High score is updated (if beaten)

---

## 💾 High Score Feature

* High score is stored in the browser using **localStorage**
* Persists even after refreshing the page

---

## 🔄 Restart

* Clicking restart:

  * Resets score
  * Resets questions
  * Restarts timer
